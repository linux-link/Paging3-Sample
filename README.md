# Paging3

> 本仓库参考了Google官方的Paging3实现：[GoogleCodeLabs-android-paging](https://github.com/googlecodelabs/android-paging)，不过为了增加其可读性，在官方demo的基础上有一定的修改。

> 配合[Android Jetpack 应用指南（一） - Paging3基础使用](https://www.jianshu.com/p/4d17a90014d6)食用更佳

## 分支说明

```
# network 分支（默认分支）
Paging3基础使用方法

# state 分支
展示如何监听Paging3的状态，重试、加载中、异常等状态

# footer 分支
展示如何使用Paging3添加尾部

#separator 分支
展示如何使用Paging3添加Item分割

# database 分支
展示Paging3如何同时从数据库、网络加载数据

```
